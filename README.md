# Crossyword
## A crossword puzzle written in Rust
<https://lukasvrenner.github.io/crossyword>

## Building
### Dependencies
- rustc
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
- cargo
    > cargo is usually installed with rustc
- wasm-bindgen-cli 
```bash
cargo install wasm-bindgen-cli
```

### Build
Run `./build.sh`
```bash
./build.sh
```
